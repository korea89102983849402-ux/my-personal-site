项目简介

这是一个基于 Google Gemini AI 的智能学习助手。用户只需输入学习目标、当前水平和可用时间，AI 就会自动生成一份详细的、分阶段的学习路线图，并推荐相关的学习资源（视频、文章等）。

技术栈

  • • React + Vite
  • • TypeScript
  • • Express (Node.js)
  • • SQLite Database
  • • Google GenAI SDK

核心功能

  • • AI 智能规划
  • • 学习进度追踪
  • • 资源自动推荐
  • • 历史记录保存
